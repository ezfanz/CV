PADUNET MOBILE MIKROTIK ROUTER SYSTEM

This system was develop to monitor heitech Mikrotik devices by tracking line 3G and 4G. This is network product implement by heitech also known as BIB (Branch In Bag). By using SDN (Software Define Networking) concept to this project flow. Currently this product are used by PNB

  • This system was develop by using bootstrap3, HTML5 and CSS for the frontend
  • The backend are using raw php 7, mysql and REST API from Mikrotik router
  • This system are integrate with custom mikrotik router and YES 4G as shown at picture above. It will detect by active IP address which is 3G and 4G. I programme the 3G and 4G by line 1 (L1) & line 2 (L2)
  • This system can monitor and edit configuration remotely from the system. user can directly console from here to see network address, enable or disable wifi ssid, change route priority, monitor device resources and enable ACL (access control list) for firewall.
  • It also have a custom reporting for uptime and downtime for the device. A bar chart shows uptime of the device by weekly and monthly. Also have a uptime and downtime report by daily, weekly, monthly and yearly in CSV.
