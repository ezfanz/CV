Wlan Log Analyzer System for ASNB

This system was develop for our client ASNB for monitoring and generate report of wifi services in their branches all in Malaysia. There are two application for this project which is landing page for wifi authentication and dashboard accounting for monitoring.

  • This system was develop by using bootstrap3, HTML5 and CSS for the frontend
  • The backend are using raw php 7, mysql and RADIUS server
  • The flow of this system are start from the router which have a RADIUS protocol to keep user details and send to the RADIUS server in mysql database then the web application will grab the data and display
  • The authentication begins when user accessing the wifi ssid. By default internet will be block.User need to authenticate first to acccessing the internet.
  • There are one WIFI user which is guest only
  • For guest, they need to register to our SMS gateway and log in using their phone number and passcode from the SMS
  • In the dashboard, there are 2 modules which is monitoring wifi session, active user and reporting in CSV and PDF
  • Reporting can generate custom by daily, weekly, monthly and yearly
