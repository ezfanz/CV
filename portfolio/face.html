FACE REGISTRATION SYSTEM FOR EVENT CSM-ACE 2018

This system was develop for Heitech participate in CSM-ACE 2018. A facial recognition by using find face engine. Im using the REST API for integrating my system with the engine. The flow of the project is, user need to register their details in heitech mobile application and get the QR code. In the QR code i grab their web services and insert to my web application. Therefore, when user insert their QR code id it will automaticallly get their details and ready to register their face. After registration done, user will walk at the main hall and greetings will appear at the screen with their name and face. My task are focus on registration face.

  • This system develop to capture users face to store in the engine database
  • The engine use is Findface facial recognition engine. My task is to develop a web appication to capture user face and parse the image to Findface database
  • By using REST API POST method, the application will capture users face and insert to Findface database
  • The raw data are in json so i need to decrypt it to array in order to store a custom variable in php
  • Language use: PHP, Javascript
  • Backend: REST API and findface engine
