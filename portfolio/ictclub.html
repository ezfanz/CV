ICT CLUB MANAGEMENT SYSTEM

This system was my final year project back when i was a diploma student. I created a management system for ICT club Polytechnic Sultan Idris Shah to keep update with events and updates of the club. This project selected top 10 best final year project among 60 team and win a consolation prize.

  • This system was develop by using bootstrap3, HTML5 and CSS for the frontend
  • The backend are using raw php 7, and mysql
  • Mostly this system are using CRUD flow which is create, register update and delete.
  • Can create album and gallery for event pictures
  • Add user, delete and update
  • Can monitor users, gallery and album
