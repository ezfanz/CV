LIBRARY MANAGEMENT SYSTEM USING ARDUINO & RFID FOR UTHM

This system develop to manage student debt or no debt when trying lending a book at library. By using arduino to java application and web application using PHP, integrate together to create a complete app

  • this system develop to manage student debt or no debt when trying lending a book at library
  • consist arduino and rfid board for the device
  • language use:
    - arduino C to capture rfid uid
    - java application to grab data from arduino
    - php for Web applications and using API to store data from java app in mysql
  • function - register
    - login authentication using rfid scanning
    - dasboard for both user & admin
    - add book, lend book
    - monitoring book, user and book lending
