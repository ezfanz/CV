MASTER ADMIN SYSTEM

This system was develop to manage our user registration, system registration, backup and data restoration, archive data, and log user and sms details.

  • This system was develop by using bootstrap3, HTML5 and CSS for the frontend
  • The backend are using raw php 7, and mysql
  • Mostly this system are using CRUD flow which is create, register update and delete.
  • There are five module in this system which is user management, system management, data backup, data restoration and log activity
  • This system are consits by two user type which is admin and operator
  • For user registration, admin need to key in the user details and system access. the user will get a notification email and needs to activate their account by set their own password. The email are sent by using our SMTP server, i just integrate the url of SMTP server with this application. The validity of this email will last long for 24 hour
  • For system management, admin need to key in a new system by insert their url and system name. By this we can monitor and keep update which system has registered in master admin
  • For data backup, operator need to select the database that need to backup in sql
  • This system also contain archive data where it can store an old data for future use
  • Data restoration also contain in this system
  • Lastly we can monitor log activity user access and sms log from wifi services user Amway and ASNB
